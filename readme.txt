1. Open Workbench and run db.sql file to create database and some dummy data
2. Extract both frontend and backend
3. Open the terminal, change directory to backend, run "npm i" to install node_modules
4. Run command "node app.js"
5. Open the terminal, change directory to frontend, run "npm i" to install node_modules
6. Run the command "npm start"

7.Open browser, type http://localhost:3000/ and press enter
